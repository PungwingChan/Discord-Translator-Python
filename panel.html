📖 Getting Started

1️⃣
Get a Bot Token

Visit the Discord Developer Portal, create an application, and copy the Bot Token.

2️⃣
Configure the Bot

Paste your Token in the "Bot Configuration" section above. Optionally set a custom Translation API endpoint.

3️⃣
Start the Service

Save your configuration, then click "Start Bot" in the "Bot Control" section.
